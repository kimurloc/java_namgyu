### 8. 반복문
#### 8.1 while문
```java
while (조건식) {
  // 조건식이 참인 동안 반복해서 실행할 구문
}
```

#### 8.2 do-while문
```java
do {
  // 최소 1회 무조건 실행 및 조건식이 참인 동안 반복해서 실행할 구문
} while (조건식); // while문 뒤에 세미클론
```

#### 8.3 for문
```java
for (초기화; 조건식; 증감식) {
  // 조건식이 참인 동안 반복해서 실행할 구문
}
```

#### 8.4 break와 continue
- `break`: 실행되는 순간 소속된 switch문이나 반복문을 종료하고 탈출
- `continue`: 이번 턴만 건너뛰고 다음 턴으로 바로 넘어감

**String[] args**
- `main` 메서드의 매개변수로, **프로그램 실행 시 전달되는 커맨드라인 인수**를 문자열 배열로 받는다.
- 인수가 없어도 배열은 `null`이 아닌 빈 배열(`length = 0`)로 전달된다.

**Math.random()**
- 0.0 이상 1.0 미만의 `double` 실수를 반환 받는 함수이다.
---
```java
int wrong = (int) Math.random() * 10;
int right = (int) (Math.random() * 10);
```
Q. 다음 두 구문의 실행 결과가 다른 이유는?

A. **`(int)`형 변환 연산도 괄호 연산**이므로 뒤의 `*`연산보다 우선순위가 높다. 0.0 이상 1.0 미만의 실수를 `int`로 형변환 시키면 값은 0이므로 wrong의 결과 값은 0.
right에서는 같은 괄호 연산이지만 왼쪽은 단순 형변환 연산자, 오른쪽은 계산 순서를 최우선으로 지정하는 괄호 연산자이므로 오른쪽을 먼저 수행 후 형변환이 실행된다. right의 결과 값은 1~9.

**Scanner 클래스**
- 사용자로부터 값을 입력 받을 때 사용되는 클래스
- `import java.util.Scanner;`구문 필요
- 사용자로부터 입력 받은 데이터는 **입력 버퍼**(`Input Buffer`)에 임시 저장되는데, 이때 `Scanner`는 공백 문자(` `, `\t`,`\n`)를 기준으로 데이터를 쪼개는데, 이떄 쪼개진 데이터의 단위를 **토큰**(`Token`)이라고 한다.
