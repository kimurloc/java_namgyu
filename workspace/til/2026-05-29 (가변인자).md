#### 10.4 가변인자(varargs, Variable Arguments)
```java
public void printSum(int... nums) {
  int sum = 0;
  for (int num : nums) {
    sum += num;
  }
  System.out.println(sum);
}
printSum(1, 2, 3); // 6
printSum(10, 20); // 30
```
- 메서드를 호출할 때 매개변수의 개수를 동적으로 지정할 수 있게 해주는 기능으로, 메서드 정의 시 타입 뒤에 세 개의 마침표(...)를 붙여 선언함
- 특징
	- **내부적으로 배열 처리** : 가변 인자로 전달받은 값들은 메서드 내부에서 해당 타입의 배열로 자동 변환되어 처리
	- **가변 인자의 위치 규칙** : 가변 인자는 메서드의 매개변수 목록에서 반드시 가장 마지막에 위치해야 함. 아닐 경우 컴파일 에러
	- **단일 가변 인자 규칙** : 하나의 메서드에는 단 하나의 가변 인자만 선언 가능
