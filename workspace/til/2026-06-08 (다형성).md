#### 12.5 Object 클래스
자바에서 생성되는 모든 클래스는 별도의 `extends` 선언이 없더라도 기본적으로 `java.lang.Object` 클래스를 최상위 부모로 상속받는다. 즉 어떤 클래스든 결국 `Object` 클래스의 자식이 되므로 **자바의 모든 객체는 `Object` 타입으로 선언이 가능**하고, `Object` 클래스는 모든 객체가 기본적으로 가져야 할 **공통 메서드를 제공**한다.
- `toString()` : 객체의 문자열 정보를 반환하는 메서드로, 기본 구현은 클래스명과 해시코드를 출력하지만 개발 시 멤버 필드 값을 출력하도록 재정의하여 디버깅에 활용
- `equals(Object obj)` : 두 객체의 동등성(내용이 같은지 여부)을 비교하는 메서드로, 기본 구현은 주소값 비교(`==`)이지만 논리적 내용 비교가 필요한 경우 재정의함
- `hashCode()` : 객체의 고유한 정수 해시코드를 반환하는 메서드이며, equals 메서드를 재정의할 때 동등한 객체는 동일한 해시코드를 반환해야 한다는 규약에 따라 함께 재정의하는 것이 관례

#### 12.6 클래스 관계 : 포함(Has-A) vs 상속(Is-A)
**포함 관계(Has-A)**
한 클래스가 다른 클래스의 인스턴스를 자신의 필드로 선언 및 소유
**상속 관계(Is-A)**
부모 클래스의 특성을 그대로 물려받아 자식 클래스로 구체화하여 설계하는 관계

### 13. 다형성과 인터페이스
#### 13.1 다형성
하나의 타입(부모 클래스나 인터페이스)으로 여러 가지 객체(자식 객체)를 참조하고 제어할 수 있는 객체지향의 핵심 특징. 즉, 동일한 호출(메서드 호출)을 하더라도 참조하는 실제 인스턴스에 따라 서로 다른 결과와 동작을 나타내는 성질을 의미
- 상속 관계 : 클래스 간 상속(`extends`)이나 인터페이스 구현(`implements`) 구조가 선언되어 있어야 함
- 메서드 오버라이딩 : 자식 클래스가 부모 클래스의 메서드를 재정의하여 자신의 기능으로 변경해 두어야 함
- 부모 타입의 참조 변수 사용: 부모 클래스나 인터페이스 타입의 변수로 자식 객체를 가리켜 제어해야 함 (업캐스팅)

#### 13.2 클래스의 업캐스팅과 다운 캐스팅
- **업캐스팅** : **자식 객체를 부모 타입**으로 형변환(자동)
- **다운캐스팅** : **부모 타입으로 형변환된 객체를 다시 원래의 자식 타입**으로 형변환하는 것 (명시적 형변환 필요)
- 실제 생성된 인스턴스 타입과 일치하지 않는 다른 자식 타입으로 잘못 다운캐스팅할 경우 `ClassCastException` 예외가 런타임에 발생하므로 주의
```java
class Animal {
  public void eat() {
    System.out.println("eat");
  }
}
class Dog extends Animal {
  public void eat() {
    System.out.println("dog eats");
  }
  public void bark() {
    System.out.println("bark");
  }
}
class Cat extends Animal {
  public void eat() {
    System.out.println("cat eats");
  }
  public void meow() {
    System.out.println("meow");
  }
}
public class Test {
  public static void main(String[] args) {
    Animal myPet = new Dog(); // upcasting, myPet instance of Dog
    myPet.eat(); // overriding, "dog eat"

    Dog namu = (Dog) myPet; // downcasting
    namu.bark(); // "bark"
    // Cat myCat = (Cat) myPet; // ClassCastException
  }
}
```

#### 13.3 instanceof 연산자
참조 변수가 실제로 어떤 타입의 객체를 가리키고 있는지 확인할 때 사용
```java
if (myPet instanceof Dog) {
  Dog namu = (Dog) myPet;
  namu.bark();
}
```
