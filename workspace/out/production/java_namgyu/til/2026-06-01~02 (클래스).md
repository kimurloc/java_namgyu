### 11. 객체지향 프로그래밍(OOP)
#### 11.1 객체, 클래스, 인스턴스
- **객체(Object)** : 고유한 속성과 기능을 가지는 유무형의 모든 실체.
- **클래스(Class)** : 객체를 정의 및 생성하기 위한 코드. 실체가 없으므로 메모리에 남지 않음
- **인스턴스(Instance)** : 클래스를 바탕으로 `new`연산자를 사용하여 `heap` 메모리에 생성된 객체
#### 11.2 클래스의 구성 요소
- **필드(Field)** : 객체의 속성을 정의한 변수
- **메서드(Method)** : 객체의 기능
- **생성자(Constructor)** : 객체가 `new`로 생성될 때 초기 세팅
```java
public class Car {
  String model; // field
  int speed;

  public Car(String model) { // constructor
    this.model = model;
    this.speed = 0;
  }

  public void accelerate() { // method
    this.speed += 10;
    System.out.println(model + " speed increased! speed : " + speed);
  }
}
```
#### 11.3 필드(Field)
- 자동 초기화: 필드는 명시적으로 초기화하지 않아도 객체가 생성될 때 각 자료형의 기본값으로 자동 초기화함.
- 기본값(배열과 동일)
	* `byte`, `short`, `int`, `long` : 0
	- `float`, `double` : 0.0
	- `boolean`: `false`
	- `char`: `'\u0000'` (공백 문자)
	- 참조 타입(`String`, 클래스 등) : `null`
#### 11.4 객체의 생성
클래스를 기반으로 `heap`메모리에 객체를 올리기 위해서 `new`연산자를 사용해야 하며, 생성된 객체의 멤버에 접근하기 위해선 `.`연산자를 사용해 접근해야한다.
- `new`연산자는 JVM에게 객체를 저장할 공간을 `heap`에 할당하도록 요청
- 할당이 완료되면 클래스에 정의된 필드들이 `heap`에 생성되고, 생성자를 호출하여 초기 설정을 마침
- 이후 생성된 객체가 위치한 `heap`영역의 참조 주소 값을 반환하여 변수에 대입
#### 11.5 생성자(Constructor)
객체가 `new` 연산자를 통해 메모리에 생성될 때 가장 먼저 호출되어 인스턴스 변수의 초기 세팅을 담당하는 특수한 메서드
- 특징
    - **반환 타입이 없음** : 일반 메서드와 달리 void조차 지정하지 않으며, 이름은 반드시 소속 클래스명과 대소문자까지 일치해야 함
    - **기본 생성자 (Default Constructor)** : 클래스 내부에 개발자가 생성자를 단 하나도 선언하지 않으면 컴파일러가 매개변수와 코드가 없는 빈 기본 생성자를 자동으로 삽입해 줌. 단, 매개변수가 있는 생성자를 하나라도 정의하면 기본 생성자는 자동으로 생성되지 않으므로 필요시 직접 명시해야 함
```java
public class Member {
  String id;

  public Member(String id) { // define constructor
    this.id = id; // initialize instance field
  }

  public static void main(String[] args) {
    Member member = new Member("user1");
    System.out.println("user id : " + member.id);
  }
}
```
#### 11.6 생성자 오버로딩
생성자 또한 매개변수의 개수나 타입을 다르게 하여 하나의 클래스 내부에 동일한 이름의 생성자를 중복 생성 가능하다.(다형성 제공)
```java
public Book() {
    this.title = "no title";
    this.author = "no author";
}

public Book(String title, String author) {
	this.title = title;
	this.author = author;
}
```

#### 11.7 this와 this()
- `this` : 생성된 자기 자신 객체를 가리키는 참조 변수. 매개변수와 필드명이 같을 때 구별하기 위해 사용
- `this()` : 같은 클래스 내의 다른 생성자를 호출할 때 사용. 생성자 내부의 첫 줄에만 작성 가능
```java
public User() {
	this("default name", 1); // 아래의 User 생성자 호출
}

public User(String name, int age) {
    this.name = name;
    this.age = age;
}
```
